# 超智能音箱项目
![图片说明](项目思维图.png)
## 项目主要实现功能
* 保留原生小爱模式
* 增加AI模式，在AI模式下可以切换不同的模型
* 增强打断模式
* 增加前缀设置
## 控制命令
## 控制命令

| 命令 | 说明 |
|------|------|
| `开启小爱` | 切换到原生小爱模式 |
| `开启AI` | 切换到 AI 模式 |
| `切换deepseek` | 切换到 DeepSeek 模型 |
| `切换openai` | 切换到 OpenAI 模型 |
| `切换谷歌` | 切换到 Gemini / Google 模型 |
| `设置前缀 主人` | 设置回复前缀，例如“主人” |
| `开启前缀` | 开启前缀播报 |
| `关闭前缀` | 关闭前缀播报 |
| `测试模型` | 测试当前模型是否正常 |
