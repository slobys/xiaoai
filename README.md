## 安装git
### Debian和Ubuntu系统
```bash
sudo apt install git -y
```
### CentOS系统
```<img width="2553" height="1182" alt="未命名文件(22)" src="https://github.com/user-attachments/assets/362ba935-1566-48f7-9c22-1d64a61bc21d" />

sudo yum install git -y
```

### 如果执行后出错，请先更新系统
* （Debian/Ubuntu系统）
```bash
sudo apt update -y
```
* （CentOS系统）
```bash
sudo yum update -y
```
