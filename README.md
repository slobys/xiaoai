# 小爱音箱接入大模型
![图片说明](项目图思维.png)
## 项目主要实现功能
* 保留原生小爱模式
* 增加AI模式，在AI模式下可以切换不同的模型
  

## 安装git
### Debian和Ubuntu系统
```bash
sudo apt install git -y
```
### CentOS系统

sudo yum install git -y
```

### 如果执行后出错，请先更新系统
* （Debian/Ubuntu系统）
```bash
sudo apt update -y
```
* （CentOS系统）
```bash
sudo yum update -y
```
## 一键脚本
```bash
sudo apt install git -y && git clone https://github.com/slobys/npm.git && cd npm && chmod +x npm.sh && ./npm.sh
```
