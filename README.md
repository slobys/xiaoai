# 超智能音箱项目
![图片说明](项目思维图.png)
## 项目主要实现功能
* 保留原生小爱模式
* 增加AI模式，在AI模式下可以切换不同的模型
  

## 安装git
### openwrt/iStoreOS系统
```bash
opkg update
opkg install git git-http ca-bundle
```
### 拉取项目
```
git clone https://github.com/slobys/xiaoai.git .
```
### [Open-XiaoAI](https://github.com/idootop/open-xiaoai)
