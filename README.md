# 超智能音箱项目
![图片说明](项目思维图.png)
## 项目主要实现功能
* 保留原生小爱模式
* 增加AI模式，在AI模式下可以切换不同的模型
* 增强打断模式
* 增加前缀设置
## 控制命令
```text
开启小爱
开启AI
切换deepseek
切换openai
切换谷歌
设置前缀 主人（红色部分可以随便设，设置之后，每次回复前，都会说这个前缀）
开启前缀
关闭前缀
测试模型（用来测试模型是否正常）
```
